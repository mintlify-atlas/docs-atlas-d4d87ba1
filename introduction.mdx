## Welcome to EUW

EUW is a specialized Frappe framework application designed to enhance work order management capabilities in ERPNext. Built with extensibility and customization in mind, EUW provides custom DocTypes, extensive field customizations, and pre-configured fixtures to streamline your work order workflows.

## What's Included

EUW provides a comprehensive set of features for work order management:

- **Custom DocTypes**: Applicability, DDS, and NRC for specialized work order tracking
- **Asset Customizations**: Enhanced asset tracking with custom fields and configurations
- **Customer Management**: Extended customer fields for better relationship management
- **Issue Tracking**: Custom issue fields for improved support and maintenance workflows
- **Item Management**: Enhanced item tracking with barcodes and custom attributes
- **Maintenance Workflows**: Purpose-driven maintenance visit configurations
